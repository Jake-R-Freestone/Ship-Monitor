# Ship Monitor

## Dependencies 
- Arrow ( Just a nicer way around datetime )
- Datetime ( Pythons standard date and time library )
- Pymongo ( Used to connect to Mongo database )
- Pytest ( Used for testing )

## Test
1. Install Dependencies 
2. Open project in terminal 
3. Run "pytest"
